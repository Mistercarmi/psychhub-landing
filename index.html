Sécurité & RGPD

🔒 Local-first

Aucun serveur externe. Vos données restent sur votre ordinateur, jamais transmises ailleurs.

📋 RGPD by design

Conformité garantie. Audit trail de toutes les modifications, aucune transmission de données patient.

🔐 Chiffrement

BitLocker sur Windows, base de données protégée, validation renforcée (IBAN, téléphone, SIRET).

🆓 Open-source

Code auditable et transparent. Communauté, pas de backdoor, confiance totale.
