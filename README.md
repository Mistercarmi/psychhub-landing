# 🧠 PsychHub
### Le cabinet organisé. Les données, chez vous.

**PsychHub** est un logiciel **local et sécurisé** pour les psychologues en cabinet solo — patients, séances, factures et KPI, sans aucun abonnement.

---

## 🚀 [Découvrir la plateforme](https://mistercarmi.github.io/psychhub-landing/)

---

## ✨ Avantages clés

### 🔒 Local-first
Vos données restent sur **votre ordinateur**. Aucun serveur externe, aucun envoi de données patient vers le cloud.

### 🚫 Sans abonnement
Gratuit et **open-source**. Une seule installation, zéro frais mensuels.

### 📱 Fonctionne partout
**PC · Tablette · Smartphone** — accessible sur tous vos appareils, même hors ligne via PWA.

### 🔗 Doctolib automatisé
**Synchronisé automatiquement** — importez vos rendez-vous Doctolib en 3 clics, sans saisie manuelle redondante.

### 🧾 Tout-en-un
- **Dossiers patients** : historique complet, notes cliniques RGPD
- **Gestion des séances** : planning, rappels automatiques, statuts à jour
- **Facturation** : génération PDF, relances automatiques, suivi des paiements
- **Tableau de bord KPI** : chiffre d'affaires, taux d'impayés, charts personnalisables
- **Éditeur WYSIWYG** : composez votre dashboard idéal en drag-drop
- **Sauvegardes automatiques** : intégrité vérifiée, backup Google Drive optionnel

### 💾 Zéro perte de données
- **Export Excel** illimité pour tous vos dossiers
- **Backup automatique** avec rotation de sauvegarde
- **Accès offline** — continuez à travailler sans connexion Internet

## 🔐 Sécurité

- ✅ **Stockage local uniquement** — vos données ne quittent jamais votre ordinateur
- ✅ **Chiffrement BitLocker** — protection du disque Windows
- ✅ **Aucun serveur tiers** — zéro transmission de données patient
- ✅ **Conformité RGPD** — audit trail de toutes les modifications
- ✅ **Open-source** — code auditable, transparent, de confiance

---

## 🛠️ Stack technique

Construit avec les technologies modernes et fiables :
- **Next.js 14** + **React 18** + **TypeScript** (strict)
- **Prisma** + **SQLite** (base de données chiffrée)
- **Tailwind CSS** + **Shadcn/ui** (interface élégante et accessible)
- **Zustand** (gestion d'état simple et performante)

---

## 📥 Installation rapide

```bash
# 1. Installer les dépendances
npm install

# 2. Configurer la base de données
npx prisma generate
npm run db:migrate

# 3. Lancer le serveur
npm run dev
# → http://localhost:3000
```
